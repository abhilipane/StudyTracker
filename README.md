# Study Tracker - Console Java Application

## Overview
Study Tracker is a console-based Java application that allows users to log daily study sessions, view summaries by date and subject, and export logs to a CSV file.

## Features
- Insert study logs (Date, Subject, Duration, Description)
- View all study logs
- Summary by Date
- Summary by Subject
- Export logs to CSV

## Technologies Used
- Core Java
- Collections Framework
- File Handling
- LocalDate (Java Time API)

## How to Run
Compile:
javac src/*.java

Run:
java -cp src StudyTrackerStarter
